Linux Debian Artifacts

Ubuntu / Kali / Raspberry Pi OS
1. Struktur Folder Debian

Setiap pakej .deb memerlukan struktur folder yang spesifik:

my-app/
├── DEBIAN/
│   └── control      # Maklumat metadata pakej
└── usr/
    └── bin/         # Destinasi fail eksekusi
    └── share/       # Ikon dan aset
2. Perintah Bina (Build Command)

Jalankan ini di terminal WSL anda untuk membina pakej:

dpkg-deb --build my-app-folder
